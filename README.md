# РакетыИнфо - Django проект о космических ракетах

## Описание

Веб-приложение для управления информацией о космических ракетах, космодромах и запусках.

## Функциональность

- **Ракеты**: просмотр, добавление, редактирование и удаление информации о ракетах
- **Космодромы**: управление данными о космодромах
- **Запуски/Миссии**: отслеживание космических запусков
- **Поиск**: глобальный поиск по всем данным
- **Админ-панель**: полное управление через Django Admin

## Модели данных

### Rocket (Ракета)
- Название, производитель, страна
- Тип (орбитальная, суборбитальная, тяжёлая, сверхтяжёлая)
- Технические характеристики (высота, диаметр, масса, полезная нагрузка)
- Статус (активная, выведена из эксплуатации, в разработке)

### Cosmodrome (Космодром)
- Название, страна, местоположение
- Год основания
- Статус активности

### Launch (Запуск)
- Название миссии
- Связь с ракетой и космодромом
- Дата и время запуска
- Статус (успешный, неудачный, частично успешный, запланирован)
- Полезная нагрузка и орбита

## Установка и запуск

1. Установите зависимости:
```bash
pip install django
```

2. Примените миграции:
```bash
python manage.py migrate
```

3. Создайте суперпользователя (для админ-панели):
```bash
python manage.py createsuperuser
```

4. Запустите сервер:
```bash
python manage.py runserver
```

5. Откройте в браузере: http://127.0.0.1:8000

## URL маршруты

- `/` - Главная страница
- `/search/` - Поиск
- `/rockets/` - Список ракет
- `/rockets/add/` - Добавить ракету
- `/cosmodromes/` - Список космодромов
- `/cosmodromes/add/` - Добавить космодром
- `/launches/` - Список запусков
- `/launches/add/` - Добавить запуск
- `/admin/` - Админ-панель Django
